# GemGem Laravel

# Requirement
 
- php 8.0 and above
- apache v2.4
- Mysql v8.0.x
- composer v2.0+


# Setup 
- clone this repo
- copy .env-example to .env
- setup database
- install composer
- install node modules

# commands 
```
    cp .env-example .env
    php artisan key:generate
    composer install
    npm install
    npm run build
    php artisan:migrate --seed
    php artisan serve
```
